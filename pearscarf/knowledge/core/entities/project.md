**project**
A named initiative, deal, integration, or campaign that people refer back to repeatedly as the subject of ongoing work. **A project is the thing work happens *for*, not the work itself.**

Extract when: the record names an effort that spans many conversations, contributors, or decisions ("the Acme API integration", "Series A raise", "Q2 launch", "the pearscarf platform").

Do not extract when:
- It's a task, ticket, issue, bug, PR, commit, or meeting — those are records of work done, not the project the work is for.
- It's a one-off request, or a feature not tracked as its own distinct effort.
- It's a code repository — not a project.
- It's a generic technology, product category, or vendor name without a named context ("Python", "Slack", "GPT").
