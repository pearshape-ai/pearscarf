**repository**
A GitHub repository — a codebase with its own issues, pull requests, branches, and contributors. Repositories are organizational units that people contribute to and reference in discussions.

Extract when: the record references a specific repo by name ("the pearscarf repo", "pushed to pearshape-ai/pearscarf"), or the record itself comes from a known repository.

Do not extract when: "repository" is used generically without naming a specific one ("we should create a new repo for this").
