## Step 1 — Create `scripts/reindex.py`

Script flow:

1. Load `.env` via `dotenv`
2. Connect to Neo4j, count existing nodes and relationships
3. Connect to Postgres, count records where `classification = 'relevant' AND indexed = TRUE`
4. Print summary: "This will delete X nodes and Y relationships from Neo4j, and reset Z records to unindexed."
5. Prompt `Continue? [y/N]` — abort on anything other than `y`
6. Run `MATCH (n) DETACH DELETE n` on Neo4j
7. Run `UPDATE records SET indexed = FALSE WHERE classification = 'relevant'` on Postgres
8. Print confirmation with counts

Uses `pearscaff.neo4j_client` and `pearscaff.db` directly. No new modules.
