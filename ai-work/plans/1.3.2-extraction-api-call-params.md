# Plan: 1.3.2 — Extraction API Call Parameters

Configure the extraction LLM call for deterministic structured output. Split system/user prompts, set temperature 0, add extraction-specific config constants.

## Steps

### Step 1: Add extraction config constants to `pearscaff/config.py`

```python
# Extraction
EXTRACTION_MODEL = os.getenv("EXTRACTION_MODEL", MODEL)
EXTRACTION_MAX_TOKENS = int(os.getenv("EXTRACTION_MAX_TOKENS", "2048"))
EXTRACTION_TEMPERATURE = 0.0
```

`EXTRACTION_MODEL` defaults to the global `MODEL` but can be overridden. Temperature is hardcoded at 0 — no reason to make it configurable.

### Step 2: Update `pearscaff/prompts/extraction.md`

Remove the user prompt section (the `---` separator and everything below it — lines 98-102). The file becomes the system prompt only: entity types, relationships, facts, what to ignore, output format.

The user prompt is fixed boilerplate — move it into the code (Step 3).

- Build user message inline in code:
