## Retriever Agent

An expert agent that communicates with the worker via session messaging. The worker sends a query, the Retriever finds relevant context, returns a structured context package.

### Three query modes, tried in sequence

```
Query comes in
    |
Step 1: Entity recognition — does the query reference a known entity?
    |
If yes -> facts lookup + graph traversal
    -> Sufficient? Return.
    |
Step 2: Vector search — embed the query, find semantically similar records in ChromaDB
    |
Assemble context package, return to worker
```

### Mode 1: Facts lookup

Direct query on facts table for a known entity:

```sql
SELECT attribute, value, source_record, updated_at 
FROM facts 
WHERE entity_id = ?
```

### Mode 2: Graph traversal

Walk edges from a known entity, up to 3 hops:

```sql
WITH RECURSIVE connected AS (
    SELECT to_entity, relationship, 0 as depth
    FROM edges WHERE from_entity = ?
    UNION ALL
    SELECT e.to_entity, e.relationship, c.depth + 1
    FROM edges e JOIN connected c ON e.from_entity = c.to_entity
    WHERE c.depth < 3
)
SELECT DISTINCT * FROM connected;
```

Fetch full records for connected record IDs from the typed tables (emails, issues).

### Mode 3: Vector search

Query ChromaDB with the embedded query text:

```python
results = collection.query(
    query_texts=[query],
    n_results=5,
    where={"classification": "relevant"}
)
```

## Communication

Worker sends request via session message:

```json
{
    "action": "retrieve_context",
    "query": "What do I know about Acme Corp?"
}
```

Retriever responds:

```json
{
    "action": "context_result",
    "facts": [
        { "entity": "Acme Corp", "attribute": "total_spend_q3", "value": "$13,000" }
    ],
    "related_records": [
        { "id": "email_71", "type": "email", "summary": "Q3 invoice, $5k", "found_via": "graph" },
        { "id": "email_88", "type": "email", "summary": "Scaling discussion", "found_via": "vector" }
    ],
    "entities": [
        { "name": "Michael Chen", "type": "person", "connection": "works_at Acme Corp" }
    ],
    "reasoning": "Found entity Acme Corp. 3 facts, 2 graph-connected records, 1 vector match."
}
```

## Trigger

Explicit only. The human asks the worker something that requires context. The worker recognizes this and calls the Retriever. Examples:

- "What do I know about Acme Corp?"
- "Brief me on Michael Chen"
- "Any emails about fundraising?"
- "What's my total spend with VendorX?"
