# Context

- Mem0 removed in 1.2.0, SQLite facts + ChromaDB restored as active storage
- Qdrant is already running in Docker from the Mem0 setup
- ChromaDB is embedded (in-process), limited filtering, designed for prototyping
- Qdrant is persistent, production-grade, and will be the vector backend for whatever memory layer comes next (Graphiti, Cognee, or custom)
- No reason to keep both — consolidate on Qdrant
