## What Gets Embedded

- The email content as-is (subject + body combined). Emails are short enough to embed directly without summarization or chunking.
- Each ChromaDB entry includes metadata for filtering:

```python
collection.add(
    ids=["email_71"],
    documents=["Subject: Q3 Invoice - $5,000\n\nHi Gev, attached is the Q3 invoice..."],
    metadatas=[{
        "type": "email",
        "source": "gmail_expert",
        "sender": "michael@acmecorp.com",
        "created_at": "2026-03-01T10:00:00Z"
    }]
)
```

## Embedding Model

- Use sentence-transformers locally (`all-MiniLM-L6-v2`) — no API dependency, no token cost, fast.
- Install: `pip install sentence-transformers`
